Solar Redesign

Desktop

About

SOLAR is the administrative system that is currently used by faculty and students at Stony Brook University. It serves a number of purposes, functioning as a class registration tool, a financial transaction service, and database of student information.

The interface faces a number of problems, however. The appearance is unappealing and outdated compared with modern UI design. It also creates confusion and frustration among many students, lacking a functional search bar, navigation menu, and sufficient mobile capabilities.

Survey


In a survey poll conducted on the SBU CSE Students Discord, Stony Brook University students rated the appearance and functionality of SOLAR on a scale of 1-5.

Results showed unfavorability towards the current system in both divisions.

For this project, I studied effective UI design on the administrative systems of other universities, and proposed a re-design using Figma for some of the website's sections.

Vertical Tab



Users instinctively start looking at the left side of the page first. Instead of creating confusion with drop down menus, vertical tabs allow for cleaner organization.

The home screen introduces the user to the application, displaying a dynamic slideshow of eye-catching photographs of the products.

Simple Icons



Icons translate words into visual objects. They easily direct the user to make the right selections by communicating an idea or action, which allows for easier navigation.

Instead of using the current pixelated, overly saturated icons, I opted for simpler designs. These icons are easy to identify, remember, and increase aesthetic appeal.

Seach Tool

For a complex system such as SOLAR, a search bar is essential for navigation. Placing it at the header ensures users can perform a search from whichever page is open.

Fonts & Colors


Sans serif fonts are crisp, clean and legible. I chose a rounded font for this design, to create an inviting feeling for the user. Instead of underlining everything, I bolded the important information. The bold draws attention to labels, such as titles and subtitles.

The white and cool-grey background colors increase clarity, legibility and frees the user from distractions.



Final Thoughts



This project allowed me to research more about UI and UX systems, especially complex systems with a lot of data. I had to identify what worked for users and what didn't. The bulk of the process required me to think about the functionality of the system, and how the interface appearance goes hand in hand with it.

The biggest challenge was determining how I could make the interface more intuitive. One of the current problems the website faces is lack of user-friendliness, so I had to consider several options to approach a solution. All in all, I learned a lot about how appearance impacts functionality through this project.
